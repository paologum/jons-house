# Jon's House - Unity 2D Game

My girlfriend's 2D virtual house - An interactive memory exploration game built with Unity.

## About

An intimate 2D top-down game where you walk through a house and interact with objects that hold special memories from our relationship. Each object tells a story - from our first date to movie nights to gaming adventures.

## Quick Start

This is a Unity project. To run it:

1. Install Unity Hub and Unity 6000.2.5f1 (or later)
2. Open this project in Unity
3. Follow the complete setup guide in [UNITY_SETUP.md](UNITY_SETUP.md)
